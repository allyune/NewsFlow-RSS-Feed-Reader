## Features
 - Supports feeds of different formats (Atom  and Rss 2.0);
 - Adding RSS feeds;
 - Listing RSS feeds with metadata;
 - Unsubscribing from the RSS feed;
 - Managing RSS feeds (bulk unsubscribe);
 - Displaying articles;
 - Reloading articles;
 - Filtering articles by publication date;
